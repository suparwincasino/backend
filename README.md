# SuperWin Casino - Backend

This repository contains the **backend** of SuperWin Casino system (Node.js + Nitro + Convex).

## Features

- Express/Nitro server
- Convex integration for Wallet & Transactions
- Game logic workflows
- Dockerized for VPS deployment
- CI/CD via GitHub Actions

## Development

```bash
# Clone repo
git clone https://github.com/yourusername/SuperWin-Casino-backend.git
cd SuperWin-Casino-backend

# Install dependencies
npm ci

# Run dev server
npm run dev
